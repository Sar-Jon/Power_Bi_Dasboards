# Data Jobs Dashboard v2 w/ Power Bi

![DashBoard Page 1](/Images/Data_Jobs_Dashboard_2.0_P1.png)

## Introduction

Finding reliable information about the data job market can be challenging, with salary data, job trends, and hiring insights often spread across multiple sources. This dashboard was created for Job Seekers, Job Transitioners, and Job Swappers who want a quick and clear view of the market.

Built using a real world dataset of 2024 data science job postings, this single page dashboard brings together key information on job titles, salaries, and locations, making it easier to explore market trends and compensation insights in one place.

## Skills Showcased

**Dashboard Design:** Created a clean and intuitive layout focused on usability and effective data storytelling.

**Power Query:** Cleaned, transformed, and prepared raw data for analysis.

**Data Modeling:** Built relationships between tables using star schema principles to support efficient reporting.

**DAX:** Created measures and calculations to generate key insights and KPIs.

**Visualizations:**

* Column, Bar, Line, and Area Charts for trend and comparison analysis
* Map Visuals for geographical insights
* Cards for highlighting key metrics
* Tables for detailed data exploration
* A variety of chart types chosen to communicate insights effectively

**Interactive Features:**

* Slicers for dynamic filtering
* Buttons and Bookmarks for seamless navigation
* Drill Through functionality for deeper analysis and exploration


## DashBoard Overview

This second version brings everything together into a single, focused dashboard, making it easier to explore the most important insights from the data job market in one place.

![DashBoard Page 1](/Images/Data_Jobs_Dashboard_2.0_P1.png)

This page provides a quick overview of key market trends and metrics. It highlights important KPIs such as **Job Count, Skills per Job, Median Yearly Salary, and Median Hourly Salary**. Users can also explore the most in demand skills, view their popularity by count or percentage, and compare salary levels across different job titles.

## Conclusion

This updated dashboard(Version 2.0), demonstrates how Power BI can transform raw job posting data into a clear and interactive tool for career analysis. By bringing key insights into a single page, users can quickly explore salary trends, in demand skills, and job opportunities within the data field. The result is a simple yet effective resource for making more informed career decisions.
