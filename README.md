## Introduction

This Power BI dashboard was created to help **job seekers, people changing careers, and professionals exploring new opportunities** better understand the data job market.

The project uses a real world dataset of 2024 data science job postings that includes job titles, salaries, and locations. The goal was to turn scattered job market information into a simple and interactive dashboard that makes trends and insights easier to understand.

## Skills Showcased
- **Data Transformation with Power Query:** Cleaned, shaped, and prepared raw data for analysis by handling missing values, changing data types, and creating new columns.
- **Measures and KPIs:** Created measures to generate insights such as Median Yearly Salary and Job Count.
- **Data Visualization:** Used column charts, bar charts, line charts, area charts, tables, cards, and map visuals to present trends and comparisons clearly.
- **Dashboard Design:** Designed a clean and easy to use layout focused on clear storytelling and user experience.
- **Interactive Reporting:** Added slicers, buttons, bookmarks, and drill through functionality to improve  navigation and allow users to explore the data in more detail.

## Conclusion
This dashboard shows how Power BI can be used to transform raw job posting data into a useful tool for career analysis. It allows users to filter, explore, and drill through data to better understand salary trends, job demand, and career opportunities in the data field.
